# JavaScript Assets

Purpose:
This directory is reserved for small progressive-enhancement scripts for the
server-rendered Intelligence Workbench.

Architecture notes:
FastAPI and Jinja remain the primary application architecture. Do not add a
front-end framework, build system, or SPA runtime here.

Future extension guidance:
Use this directory only for focused interactions such as keyboard navigation,
drawer toggles, or table affordances that cannot be handled cleanly with HTML
and CSS.
