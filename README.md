# Trend Intelligence System

A personal project for collecting and organizing technology, AI, startup, and cybersecurity news from RSS feeds.

## Current Features

* RSS feed collection
* Feed parsing
* CSV storage
* Duplicate detection

## Sources

* The Hacker News
* Bleeping Computer
* TechCrunch Startups
* Hugging Face Blog

## Workflow

RSS Feeds → Feed Parser → Duplicate Check → articles.csv

## Technologies

* Python
* Feedparser
* CSV
* Pathlib

## Roadmap

* AI Summaries
* Topic Classification
* Trend Detection
* Automated Reports
